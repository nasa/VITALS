How to: Access and Work with ECOSTRESS Data

Summary

This notebook shows how to access ECOsystem Spaceborne Thermal Radiometer Experiment on Space Station (ECOSTRESS) data programmatically using the earthaccess python library leveraging NASA’s Common Metadata Repository (CMR) and enabling authentication, searching, downloading, and streaming of data with minimal coding. It also shows how to work with ECOSTRESS Tiled Land Surface Temperature and Emissivity (ECOSTRESS_L2T_LSTE) product hosted in the cloud and managed by the Land Processes Distributed Active Archive Center (LP DAAC).

Learning Objectives

  • How to search ECOSTRESS data using earthaccess
  • How to stream or download ECOSTRESS data
  • How to clip ECOSTRESS data to a Region of Interest (ROI)
  • How to quality filter ECOSTRESS data
  • How to export the processed ECOSTRESS data

Requirements

  • NASA Earthdata Login account. If you do not have an Earthdata Account, you can create one here.

Authentication

Log into Earthdata using the Auth and login functions from the earthaccess library. The persist=True argument will create a local .netrc file if it doesn’t exist, or add your login info to an existing .netrc file. If no Earthdata Login credentials are found in the .netrc you’ll be prompted for them.

auth = earthaccess.login(persist = True)
# are we authenticated?
print(auth.authenticated)
True

Setting Up Our Study Area

For this tutorial, we’ll study land surface temperature around Boulder, Colorado - a great example for examining urban heat patterns in a mountain setting.

# Create a local directory for our data
data_dir = './ecostress_tutorial_data/'
if not os.path.exists(data_dir):
    os.makedirs(data_dir, exist_ok=True)
    print(f"📁 Created data directory: {data_dir}")

# Define our study area: Boulder, Colorado
# These coordinates define a bounding box around Boulder
boulder_bbox = (-105.301, 39.957, -105.178, 40.094)
print(f"🎯 Study area: Boulder, Colorado")
print(f"📍 Bounding box: {boulder_bbox}")

# Create a simple polygon for our study area (for visualization)
bbox_geom = box(*boulder_bbox)
study_area = gp.GeoDataFrame([1], geometry=[bbox_geom], crs='EPSG:4326')
study_area['name'] = 'Boulder, CO'

# Let's visualize our study area
study_area.hvplot(
    tiles='ESRI', color='red', alpha=0.3, line_color='red', line_width=2,
    frame_height=400, frame_width=600, title='Study Area: Boulder, Colorado'
)

Searching for ECOSTRESS Data

Now let’s search for ECOSTRESS land surface temperature data over our study area. ECOSTRESS provides high-resolution thermal data that’s perfect for studying urban heat islands.

# Search for ECOSTRESS data
print("🔍 Searching for ECOSTRESS data...")

results = earthaccess.search_data(
    provider='LPCLOUD',                    # NASA's cloud data provider
    short_name='ECO_L2T_LSTE',            # ECOSTRESS Land Surface Temperature & Emissivity
    version='002',                         # Product version
    bounding_box=boulder_bbox,             # Our Boulder study area
    temporal=('2023-07-01', '2023-08-01'), # Summer 2023 - peak heat season
    count=50                               # Maximum number of scenes to find
)

print(f"🛰️ Found {len(results)} ECOSTRESS scenes over Boulder!")

# Let's examine what we found
if results:
    print(f"📅 Date range: {results[0]['umm']['TemporalExtent']['RangeDateTime']['BeginningDateTime'][:10]} to {results[-1]['umm']['TemporalExtent']['RangeDateTime']['BeginningDateTime'][:10]}")
else:
    print("❌ No data found. Try adjusting your search parameters.")

Understanding ECOSTRESS Data Structure

ECOSTRESS data comes in tiles with multiple layers. Let’s extract the links to the specific data we need:

# Extract download links for Land Surface Temperature (LST) and Quality Control (QC) data
lst_links = [link for scene in results for link in scene.data_links() if 'LST.tif' in link]
qc_links = [link for scene in results for link in scene.data_links() if 'QC.tif' in link]

print(f"🌡️ Found {len(lst_links)} Land Surface Temperature files")
print(f"🔍 Found {len(qc_links)} Quality Control files")

if lst_links:
    print(f"\n📋 Example filename breakdown:")
    example_filename = lst_links[0].split('/')[-1]
    print(f"   {example_filename}")
    print(f"   ECOv002: ECOSTRESS version 002")
    print(f"   L2T: Level 2 Tiled product")
    print(f"   LSTE: Land Surface Temperature & Emissivity")
    print(f"   Date/Time: Acquisition date and time in UTC")
    print(f"   LST.tif: Land Surface Temperature layer")

Accessing and Visualizing ECOSTRESS Data

Now for the exciting part - let’s access and visualize the satellite thermal data! We’ll stream the data directly from NASA’s cloud without downloading large files.

if lst_links:
    print("🌐 Streaming ECOSTRESS data from NASA's cloud...")
    
    # Configure rasterio for cloud access
    rio_env = rio.Env(
        GDAL_DISABLE_READDIR_ON_OPEN='EMPTY_DIR',
        GDAL_HTTP_COOKIEFILE=os.path.expanduser('~/cookies.txt'),
        GDAL_HTTP_COOKIEJAR=os.path.expanduser('~/cookies.txt')
    )
    rio_env.__enter__()
    
    # Select a representative scene (middle of our time series if available)
    scene_index = min(len(lst_links)//2, len(lst_links)-1)
    selected_scene = lst_links[scene_index]
    
    print(f"📡 Loading scene: {selected_scene.split('/')[-1]}")
    
    # Stream the data directly from NASA's cloud
    eco_lst = rxr.open_rasterio(selected_scene).squeeze('band', drop=True)
    
    print(f"✅ Data loaded successfully!")
    print(f"📊 Data shape: {eco_lst.shape}")
    print(f"🌡️ Temperature range: {float(eco_lst.min().values):.1f}K to {float(eco_lst.max().values):.1f}K")
    print(f"🔥 That's {float(eco_lst.min().values)-273.15:.1f}°C to {float(eco_lst.max().values)-273.15:.1f}°C")
    
    # Display basic information about our data
    print(f"\n📋 Dataset Information:")
    print(eco_lst)

else:
    print("❌ No LST data available for visualization")

Creating an Interactive Visualization

Let’s create an engaging, interactive map showing land surface temperature over Boulder:

if 'eco_lst' in locals():
    print("🎨 Creating interactive temperature map...")
    
    # First, let's check our data
    print(f"📊 Data shape: {eco_lst.shape}")
    print(f"📍 Data CRS: {eco_lst.rio.crs}")
    print(f"🌡️ Temperature range: {float(eco_lst.min().values):.1f}K to {float(eco_lst.max().values):.1f}K")
    
    # Reproject to geographic coordinates for visualization
    eco_lst_geo = eco_lst.rio.reproject('EPSG:4326')
    print("✅ Data reprojected to geographic coordinates")
    
    # Plotting configuration for an attractive visualization
    plot_opts = dict(
        frame_height=500, 
        frame_width=700, 
        fontscale=1.2,
        cmap='RdYlBu_r',  # Red-Yellow-Blue colormap (reversed so red=hot)
        colorbar=True,
        tools=['hover'],  # Add hover tool for interactivity
    )
    
    # Determine good color limits based on actual data
    temp_min = float(eco_lst_geo.quantile(0.02))  # 2nd percentile
    temp_max = float(eco_lst_geo.quantile(0.98))  # 98th percentile
    plot_opts['clim'] = (temp_min, temp_max)
    
    print(f"🎨 Using temperature range: {temp_min:.1f}K to {temp_max:.1f}K for visualization")
    
    try:
        # Create the main temperature plot
        temp_plot = eco_lst_geo.hvplot.image(
            x='x', y='y', 
            tiles='ESRI',  # Satellite imagery background
            xlabel='Longitude', 
            ylabel='Latitude', 
            title='🌡️ Land Surface Temperature - Boulder, Colorado\n(ECOSTRESS Satellite Data)',
            crs='EPSG:4326',
            alpha=0.8,  # Slight transparency to see background
            **plot_opts
        )
        
        # Add study area boundary
        boundary_plot = study_area.hvplot(
            color='white', 
            alpha=0.0,  # Make fill transparent
            line_color='black', 
            line_width=3,
            crs='EPSG:4326'
        )
        
        # Combine the plots
        final_plot = temp_plot * boundary_plot
        
        print("🎉 Interactive map created! Look for hot spots (red) and cool areas (blue)")
        print("💡 Tips: Urban areas are typically hotter than vegetation and water bodies")
        
        # Try to display the plot
        try:
            # For Jupyter notebooks
            final_plot.show()
        except:
            # Alternative display method
            print("📺 Attempting to display plot...")
            display(final_plot)
        
    except Exception as e:
        print(f"❌ Error creating visualization: {e}")
        print("🔧 Let's try a simpler plot...")
        
        # Fallback: simpler matplotlib-style plot
        import matplotlib.pyplot as plt
        
        fig, ax = plt.subplots(figsize=(10, 8))
        eco_lst_geo.plot(ax=ax, cmap='RdYlBu_r', cbar_kwargs={'label': 'Temperature (K)'})
        
        # Add study area boundary
        study_area.boundary.plot(ax=ax, color='black', linewidth=2)
        
        ax.set_title('🌡️ Land Surface Temperature - Boulder, Colorado\n(ECOSTRESS Satellite Data)')
        ax.set_xlabel('Longitude')
        ax.set_ylabel('Latitude')
        
        plt.tight_layout()
        plt.show()
        
else:
    print("⚠️ No temperature data available for plotting")
    print("🔍 Make sure the previous data loading step completed successfully")
